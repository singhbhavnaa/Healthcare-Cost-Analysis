# Healthcare Cost Drivers Analysis & Predictive Modelling

## Project Overview
This project analyses medical insurance data to identify key factors influencing healthcare costs. The goal is to support insurance companies in pricing strategy, risk assessment, and cost optimisation using data analytics and machine learning.

## Business Problem
Insurance providers need to understand what drives high medical expenses in order to:

- Improve premium pricing strategies
- Identify high-risk patient segments
- Optimise financial forecasting
- Enhance decision-making using data insights

## Tools & Technologies
- Python (Pandas, NumPy, Scikit-Learn)
- Data Visualisation (Matplotlib, Seaborn)
- Machine Learning (Linear Regression, Random Forest)
- Statistical Analysis
- Power BI (Dashboard)

## Key Analysis Performed
- Exploratory Data Analysis (EDA)
- Cost distribution analysis
- Smoking impact on medical charges
- BMI risk segmentation
- Regional healthcare cost comparison
- Feature engineering (BMI category)

## Machine Learning Models
- Linear Regression
- Random Forest Regressor

## Key Insights
- Smoking significantly increases healthcare expenses
- Higher BMI correlates with higher medical costs
- Age is a strong predictor of insurance charges
- Machine learning can effectively predict insurance cost trends

## Business Impact
This analysis helps insurance companies:

- Identify high-risk customer groups
- Optimise pricing models
- Reduce financial risk
- Enable data-driven strategic decisions

## Project Structure
- Notebook → Data analysis & modeling
- Dataset → Insurance cost dataset
- Dashboard → Power BI visualisation
